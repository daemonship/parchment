# TTRPG Handout Generator: Customizable Player Handouts from Text

> Game Masters spend hours making in-world documents (letters, posters, spellbooks) for tabletop RPGs. Generic template tools lack genre-specific styling.

## What It Solves

Game Masters spend hours making in-world documents (letters, posters, spellbooks) for tabletop RPGs. Generic template tools lack genre-specific styling.

## Who It's For

Dungeon Masters (D&D), Game Masters for systems like Call of Cthulhu or Cyberpunk.

## Tech Stack
- **Frontend:** Svelte 5 + TypeScript + Vite
- **Export:** html2canvas for PNG generation
- **Deployment:** Vercel/Netlify ready
